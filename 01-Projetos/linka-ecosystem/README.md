## 🎯 Visão do Produto

LINKA é uma plataforma/marca regional voltada para:
- Agregação e descoberta de eventos → **RadarLinka**
- Presença web da marca → **LinkaSite**
- Ferramentas para representantes/distribuidores → **Mapa Trigo**
- Conteúdo periódico de IA → **promptSemanais**

---

## 📐 Stack Padrão do Ecossistema

- **Frontend web:** React + Vite + Tailwind (projetos maiores) / HTML puro (utilitários)
- **Backend/automação:** Python
- **AI:** Groq API (llama-3.3-70b)
- **Deploy:** Vercel (todos)

---

## 🔑 Contexto

- RadarLinka é o produto mais maduro — pipeline completo funcionando em produção
- LinkaSite está em early stage (poucos commits)
- Mapa Trigo: interface `multimin-representantes-preview.html` com mapa geográfico de representantes
- promptSemanais: conteúdo de IA publicado semanalmente, stack simples HTML/JS
